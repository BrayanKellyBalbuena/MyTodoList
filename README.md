# My Todo List

Application that allows you to manage your task list.

## ScreenShots

<img src="ScreenShots/mylist.gif" />

## Actual Features
* Create category list of todo
* Add new todo to category
* Remove Categories
* Remove todo items
* Search category and todo items by name
* Save data to local device
